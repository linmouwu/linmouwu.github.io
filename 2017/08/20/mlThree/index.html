分类的定义

分类，即clasification，是根据观测值推算预测结果所属类型的过程。分类问题与回归问题很类似，均属于监督学习问题。区别在于，回归问题预测的是连续数值，而分类问题预测的是离散的结果。根据分类结果的类别数量，我们可以把问题定义为：0-1分类，多级分类等。

逻辑回归方程（Sigmoid Function）

逻辑回归方程，Logistic Function，也叫Sigmoid Function，是根据训练集所构建起来的预测模型。具体形式大致为：

  • h_theta(x) = g(theta’ * x)
  • z = theta’ * x
  • g(z) = 1 / (1 + e^-z)
    这里的h表示的是输出为1的可能性，0 <= h <= 1，sigmoid function g(z) 的值域也是0到1。

Decision Boundary

中文翻译‘判定边界’，是区分逻辑回归结果对应的不同类的界线。例如对于0-1分类，则是y＝0和y＝1之间的界线。
对于sigmoid方程，我们有：

  • z = 0，g(z) = 0.5； z = -INIF， g(z) = 0； z = +INIF， g(z) = 1；
  • theta’ x >= 0 则 y = 1； theta’ x < 0 则 y = 0。

Cost Function

价值方程（一直忍不住翻译成‘花费方程’）。通用形式和线性回归类似，也是预测值和实际值的差的平方和的均值。由于h在0到1的区间，所以对于越接近y的h（毕竟在这里，y是｛0，1｝而h是连续函数），其惩罚应该越小。

  • if y = 1， Cost(h, y) = -log(h)
  • if y = 0， Cost(h, y) = -log(1 - h)
  • 简化之后得到，Cost(h, y) = -y log(h) - (1 - y) log(1 - h)
  • 将其并入最终的价值方程，得到J(theta) ＝ (-y’ log(h) - (1 - y)’ log(1 - h)) / m
  • 对于theta求导之后，得到梯度递减函数为：thetaj = thetaj - alpha/m sum((h(xi) - yi) xi)， 也可向量化成theta = theta - alpha/m X’ (g(X * theta) - y)

多级分类问题

简单的说就是运用0-1分类问题的思想，区别在于y = ｛0，1，2，3，…n｝，且我们需要分别求得

  • h0(x)，h1(x)，h2(x)，h3(x)等等
  • 取得hi(x)得最大值

拟合不足和过度拟合

拟合不足：训练出来的模型甚至不能很好得预测训练集的结果。价值方程结果过大。
拟合不足解决方法：

  1. 调参
  2. 修改模型幂级
    过度拟合：训练出来的模型过分拟合训练集。价值方程过分理想。
    过度拟合解决方案：
  3. 降维（减少特征数量）
  4. 规范化theta（降其下降至相近的magnitude）

规范化theta

目的是为了平衡不同thetaj之间的权重。做法为是在Cost Function中加入额外的规范化参数项。

  • J(theta) = 1/2m (sumOfEachCost^2 + lamda sum(theta^2))。
    但是这里的lamda需要很小心，过大的lamda会导致拟合不足的问题。
    拟合之后，回归方程的梯度下降形式有所改变：
  • theta需要额外减去一项 alpha lamda / m theta。提取出theta会发现theta乘以（1 － alpha * lamda / m），所以theta不断递减。
  • Normal Equation：theta ＝ pinv(X’ X + lamda L) X’ y， L为斜对角线除了（0，0）全为1，其他元素全为0的矩阵。
