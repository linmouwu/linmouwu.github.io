Matlab Introduction

常见的Matlab命令和函数：

  • A = [1, 2, 3]是向量Vector，向量是竖向的；A(i)表示第i个元素，从1开始；
  • length(A)表示向量的长度；
  • B = [1, 2, 3; 4, 5, 6]是2*3矩阵matrix，行数row为2，列数column为3；B(i, j)表示第（i,j）个元素，从1开始，可以用:表示从第几行(列)到第几行(列)；
  • size(B, x)表示矩阵的长或者宽，x = 1 为宽， x ＝ 2 为长；
  • B’ 表示矩阵的转制， pinv(B)表示求矩阵的秩；
  • X可以为训练集中feature组成的矩阵，每一行为一个training point；theta为feature的参数；用X ＊ theta是一种vectorization的手段，结果是预测的h(X).
  • function [A, B] = myFunction(Arg1, Argu2) 用于定义函数，A、 B为返回值；

多变量线性回归

多变量线性回归，即多元线性回归，是一元线性回归的推广。对于每一个训练集xi，都包含了多个变量xj。这里的i的从1到m，m是训练集的大小；而j从0到n，n是变量的多少，其中，x0 ＝ 1.

  • 注意，vectorization对于多元线性回归很有用：h(x) = X * theta.
  • Gradient descent公式：theta ＝ theta － alpha/m X sum（X * theta － y）

Feature Scaling和Mean normalization

直译就是“特征缩放”和“均值标准化”。这两者都有利于提高梯度下降收敛的速度。当不同的特征落在相近的范围时，梯度下降的方向更加趋向与最低点；若不是，则剃度下降在数值范围比较大的方向上变化会影响数值范围比较小的方向上的变化。

  • 特征缩放：本来的数值减去均值，再除以范围（最大最小值的差）；结果为－1到1；
  • 均值标准化：本来的数值减去均值，再除以标准差；结果的均值为0；

判断alpha值选择的是否恰当

以迭代次数为x轴，要最小化的cost function J作为y轴，在坐标系中画出J的变化曲线。如果J单调递减，说明迭代次数越多，J越小，alpha选择正确。

  • 如果J先减小后增大，说明alpha选择过大；
  • 如果J减小速度太慢，说明alpha选择过小；
  • 可以在图中选择一个数值表示J值收敛的阈值。

多项式回归

对于特征值xj，我们除了可以引用其直接的一次形式，还可以引用其多项式形式，这要会导致线性回归方程的回归曲线成为非直线。

正常方程

正常方程，是在向量化特征值、theta值和预测值之后，通过矩阵运算而实现对theta向量的求解的方程式。具体是：

theta = pinv(X’ X) X’ * y

  • 当n（特征数量）趋于无限大时，选择梯度下降，计算复杂度为O（kn^2）
  • 当n比较小，且不想迭代运算时，选择正常方程，复杂度为O（n^3）
  • 正常方程有局限性，当X’ * X不可逆，即无法求的其秩的时候（此时X中可能有冗余特征，或者太多特征），则正常方程只能给出最优结果的近似值。
