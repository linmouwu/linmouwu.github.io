Introduction

主要介绍了课程本身和几个ML常见的定义。总（fan）结（yi）如下：

  • 机器学习（Machine Learning）：机器学习是研究“如何让一个机器或者程序在自发学习而无人为干涉的前提下，给出实际问题一个可行答案”的学科。按照Tom Mitchell的定义，可具体化为：一个计算机程序在经验E中学习，任务T是学习所要的结果，而性能P则是学习过程和结果的度量。机器学习的分支与应用包括：人工智能，深度学习，数据挖掘，语音识别，图像识别等等。
  • 监督学习（Supervised Learning）：在给定已有数据集的情况下，找到未知量与已知参数之间关系的机器学习范畴。已有数据集称为“训练集”。监督学习依照学习目的的不同，大致可以分为“回归（regression）”和“分类（classification）”两种问题。
  • 非监督学习（Unsupervised Learning）：在没有已知数据集的情况下，找到数据集本身特性的机器学习范畴。非监督学习不含训练集，因此其目的主要是发现数据集本身的特性、或数据直接本身所具有的关系。主要应用包括“聚类（clustering）”等问题。

模型和成本函数

主要从宏观上介绍机器学习的实际过程：

  • 模型：监督学习的模型即学习的结果，用来表示预测变量（标签label）和给定已知变量之间的关系。预测变量实际值为y，已知变量为xi，模型本身为h，则我们需要找到的关系就是找到最理想的h，是的满足h（xi）（h（xi）表示预测变量的预测值）与实际的y之间的差值最小；差值越小，说明预测越准确。
  • 成本函数（cost function）：用来表示预测变量的预测值和实际值之间偏差的函数，即模型h的准确程度的函数。常见的指标是J＝“预测值与实际值的差的平方的均值”。即：J＝1/2*m＊sum［（h（xi）－y）^2］。这里的m表示测试数据集的大小；除以2是为了梯度下降算法的计算方便。另外，拟合程度达到完美时，J＝0。
  • 轮廓图（contour plot）：坐标轴为二维，以轮廓线作为第三维的图像；同一轮廓线上的值相同，轮廓线越密集，斜率越大。轮廓图可以用于寻找拟合曲线最好的点。对于成本函数而言，轮廓图的最低点就是成本函数的最小点。

梯度下降

主要介绍梯度下降算法：

  • 梯度下降（gradient descent）：这是一种不断逼近曲线或者曲面极小值的方法。其做法是在每一步的时候，不断更新自变量，更新的幅度为当前的导数（derivative）乘以当前的因变量，直到满足收敛条件为止。具体的公式为：thj：＝thj－arf＊derivative（J）。其中，derivative是J对应于第j个自变量的偏导。而且应该注意，所有的thj都同时改变而不是依次改变。
  • 梯度下降之所以能成功的一个原因是：负号和偏导的存在，使得当前的变量的点会往极值点移动：例如thj<peek，则偏导为负，那么thj增加；反之亦然。而arf的存在用来控制变化的步长。当arf过大，thj变化过于明显，可能出现thj在极值点周围来回跳动的情况，甚至可能导致thj无法收敛；当arf过小，则步长太短，收敛速度过于缓慢。

矩阵运算回顾：

主要回顾了常见的矩阵运算：

  • 加法：矩阵的行列数相同，直接加减；
  • 乘法：(mn) (np) = (mp)，最后行数和第一个矩阵相同，列数和第二个矩阵相同，所以第一个矩阵的行数越少，第二个矩阵的列数越少，越节约空间；
  • 向量（Vector）：单一一列，多行；
  • 矩阵的逆（inverse）：A*A^-1=I；
  • 矩阵的转置（transpose）：沿着左上－右下斜对角线对称。
