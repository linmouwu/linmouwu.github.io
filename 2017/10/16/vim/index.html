Vim 小抄

发表于 10/16/2017 | 分类于 计算机基础

Manager是个Vim的狂热粉丝，每次看他在Linux下快速敲敲打打都觉得很溜……所以稍微google了一些Vim常用的short－cut。希望以后用得上吧（其实不希望用得上……

简述

Vim是Linux下的文本编辑器，分为Insert Mode，Normal Mode和Visual Mode。细节略。

基本操作

  • h：左
  • j：下
  • k：上
  • l：右
  • w：单词头
  • e：前一个单词尾
  • b：单词尾
  • $：行末
  • 0：行首
  • ^：行首的第一个非空格
  • {：上一个代码块
  • }：下一个代码块

页面定位

  • gg：文本的第一行开头
  • G：文本的最后一行开头
  • H：当前页面的第一行开头
  • M：当前页面的中间开头
  • L：当前页面的最后一行开头

屏幕滚动

  • Ctrl + f：forward一个屏幕
  • Ctrl + b：backward一个屏幕
  • Ctrl + d：forward半个屏幕
  • Ctrl + u：backward半个屏幕
  • Ctrl + e：forward一行
  • Ctrl + y：backward一行

编辑模式

  • I：在行首插入
  • A：在行末插入
  • a：光标的后一个字符插入
  • o：上一行插入
  • O：下一行插入

查找

  • /pattern：正向查找
  • ?pattern：反向查找
  • n：下一个匹配
  • N：上一个匹配
  • :%s/old/new/g：new替换old

剪切、复制、粘贴

  • dd：剪切当前行
  • D：剪切到行末
  • yy：复制当前行
  • p：粘贴到光标后
  • u：撤销

组合操作

  • v(选择),c(插入),d(剪切),y(复制) ＋ ‘i/a’ + ‘b/B/\’/\”‘
  • gd：跳转到函数定义
