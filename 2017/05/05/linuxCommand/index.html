常用Linux命令

发表于 05/05/2017 | 分类于 计算机基础

Petuum尚在继续！革命仍未结束！既是为了Backend Engineer做准备，也是为了以后工作打基础。要是学快5-6年计算机，连常见的Linux命令都整不清楚，有点说不通吧。
废话少说，我们开始。

主要参考资料

The Linux Command Line

Navigation

  • pwd 当前目录绝对路径
  • cd 改变目录
  • ls 列出当前目录的内容
  • . 当前路径
  • .. 绝对路径
  • ~ 用户目录
  • / 根目录

System

  • ls: -l 详细信息， -h 人类可读
  • file: 文件类型
  • less: 查看文件内容，提供分页、跳转、查找功能
  • more: 查看文件内容，提供分页、继续功能
  • cat: 查看文件内容，提供链接功能

Files and Directories

  • cp source target: 复制
  • mv source target: 移动
  • rm: 移除
  • mkdir: 创建文件夹

Commands for Commands

  • type: 命令类型
  • info: 命令信息
  • man: 命令主页
  • alias: 命令别名
  • help: 命令帮助

Redirection

  • sort: 文本排序
  • uniq: 除重
  • grep: 输出匹配文本
  • head: 文件头
  • tail: 文件尾
  • >: 输出重定向
  • >>: 输出append到重定向
  • |: 管道，第一个命令的输出作为第二个命令的输入

Permission

  • su: 其它用户
  • sudo: 根用户
  • chmod: 修改文件权限
  • passwd: 修改密码
  • rwx: 读，写，执行
  • 777: 拥有者，用户群，全局

Process

  • ps: 当前进程
  • kill: 杀死进程
  • jobs: 任务列表
  • bg: 前台
  • fg: 后台

Text Editor - vi, vim

  • :q 退出
  • :w 写入
  • :i 键入
  • yy 复制
  • p 粘贴
  • dd 删除行
  • x 删除光标当前字符
  • /str 搜索str
