神经网络的Cost Function

  • L = total number of layers in the network
  • sl = numbers of units (excluding bias) in layer l
  • K = number of output units/classes
  • j = the jth node in layer l+1
  • i = the ith node in layer l
  • htheta(x)k = result of kth output
    Cost function：很复杂，大致就是所有输出类的差值的和加上regularized引入的所有layer的所有units的所有parameter的theta的平方和
  • J = - (1/m) sumOf(sumOf(yk log(h(x)k) + (1 - yk) log(1 - h(x)k))) + (lambda/2m) sumOfL(sumOfi(sumOfj(thetaj,i,L)));
  • 神经网络中，对于下一层的每一个node，我们分别用这一层的所有node乘以对应的theta并求和得出的。所以当前的theta matrix中，theta的行数等于下一个layer的node数量（不包括bias）；theta的列数等于当前layer的node数量（包括bias）；
  • 求regularized的参数的时候，需要把所有层与层之间的所有theta的平方求和。

逆向传播算法

逆向传播算法是神经网络中用来最小化cost function的方法（即gradient descend的替代），其最终输出为grad，就是优化参数。其核心还是对于所有theta的偏导的合理化。为了实现最小化，我们需要先找出所有的偏导函数。
具体算法过程：

  1. 初始化所有的deltaijl为0，deltaijl表示计算到当前样本p时的Dijl的偏导的初始形式。偏导推算的数学过程就无视好了，因为不会。
  2. 每次取一个测试样本（xp, yp），p表示pth样本，重复第3 － 7步；
  3. 将第一层的节点a1初始化为xp；
  4. 不断计算每一层的节点al，计算方式为逻辑推理运算，即a(l+1)j ＝ g（thetalj * al）；
  5. 计算出最终的h（xp），即aL，并求得当前的deltaL ＝ aL - yp；
  6. 计算deltal = (thetal’ delta(l+1)) . al . (1 - al)，求出所有layer的所有deltal；这里我们知道g’(zl) = al . (1 - al)；
  7. 计算deltaijl = deltaijl + ajl * deltaLi(l+1)；
  8. 第3 － 7步找出了对应于第p个测试样本的deltaijl的部分值；
  9. 对于最终的Dijl，Dijl表示第ijl个theta的偏导函数，我们有Dijl = 1/m (deltaijl + lambda thetaijl)，其中当j＝0，即表示下一层的bias的时候，lambda等于0，不需要regularized。
  10. 在上述步骤中，deltal其实是第l层的节点错误的偏导；
  11. 得到偏导函数后，我们就可以不断训练theta值从而获得最接近于0的偏导值，从而获得最小的cost function；

Unroll matrix

将矩阵重写成长的列向量。实例：

  • thetaVector = [ Theta1(:); Theta2(:); Theta3(:); ]
  • deltaVector = [ D1(:); D2(:); D3(:) ]

Gradient checking

就是偏导函数（梯度函数）的检查，目的是为了保证梯度函数实现的正确性。
简单的说，就是求一下J（theta）在theta点左右epsilon的范围内的值，(J(theta + epsilon) - J(theta - epsilon))/ 2epsilon，看其是否极为我们的grad的值；

Random Initialization

初始化theta为非0但是无限小的值。

Training a Neural Network

下面这一段表示的是generally怎么训练一个神经网络。因为跟上文其实差不多，所以就不重复翻译，直接贴过来（由此证明笔者本人有多懒。你管我。）

  • Randomly initialize the weights
  • Implement forward propagation to get hΘ(x(i)) for any x(i)
  • Implement the cost function
  • Implement backpropagation to compute partial derivatives
  • Use gradient checking to confirm that your backpropagation works. Then disable gradient checking.
  • Use gradient descent or a built-in optimization function to minimize the cost function with the weights in theta.
